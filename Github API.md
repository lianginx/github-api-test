### 记录

> Base：https://api.github.com/

| 功能        | 路径                             |
| :---------- | :------------------------------- |
| 用户信息    | /users/用户名                    |
| 仓库列表    | /users/用户名/repos              |
| Issues 列表 | /repos/用户名/仓库名/issues      |
| Issues 详情 | /repos/用户名/仓库名/issues/序号 |
